Status: `PENDING_OWNER_RUN_RECORDING`

This backlog is seeded for the first signed-user recording. Items should be promoted only when supported by video, screenshots, route timings, console errors, network failures, or owner-observed friction.

## P0 Launch / Revenue Blockers

- Any `/reader/dracula` "Reader not found" state for a signed user.
- Any public Listen Now CTA, AudioObject metadata, public audio URL, or audiobook-live claim.
- Any Kshudhita or pipeline title receiving Start Reading, payment, or audio CTAs.
- Any live Razorpay checkout execution triggered by the recorder.
- Any admin/launch-monitor data exposed to non-admin users.
- Any route blank screen on core revenue paths: `/`, `/book/dracula`, `/reader/dracula`, `/pricing`, `/account`.

## P1 Premium UX Improvements

- Hero CTA hierarchy confusion or below-fold mobile CTA placement.
- Site-tour copy/focus issues.
- Library pipeline CTAs that feel mysterious or non-actionable.
- Reader typography, width, contrast, spacing, or scroll comfort issues.
- Pricing or wallet language that does not quickly explain reading time.
- Missing or broken social link affordances.
- Admin dashboard empty states that feel broken rather than intentionally empty.
