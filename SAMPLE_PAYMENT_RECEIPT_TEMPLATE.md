# Sample Payment Receipt Template

Status: `SAMPLE_ONLY_NOT_TAX_INVOICE`

This is a sample payment receipt format for owner review. It contains no real customer data, no full Razorpay IDs, no card/UPI/bank details, and no secrets.

Important: This is a payment receipt template, not a tax invoice unless explicitly marked as a tax invoice after tax compliance is implemented.

## Sample Receipt

| Field | Sample Value |
| --- | --- |
| Merchant | Earnalism / REO Enterprise Venture |
| Receipt number | EARN-REDACTED-SAMPLE-0001 |
| Date/time | YYYY-MM-DD HH:MM IST |
| Customer | Reader account ending in `[redacted]` |
| Item | Earnalism reading-time wallet top-up / 7-day reading pass |
| Amount | REDACTED_LOW_VALUE_AMOUNT |
| Currency | INR |
| Payment status | PAID |
| Payment provider | Razorpay |
| Redacted payment reference | RZP-REDACTED-REF |
| Wallet credit | REDACTED_READING_TIME_CREDIT |
| Support contact | Earnalism support channel |

## Disclaimer

This is a payment receipt, not a tax invoice unless explicitly marked as a tax invoice. This sample does not prove that automated customer receipt email or GST invoice generation is enabled.

## Data Safety

- No real customer email.
- No phone number.
- No payment ID.
- No order ID.
- No customer ID.
- No card, UPI, or bank details.
- No webhook secret.
- No API key.
- No invoice number from a real accounting system.
