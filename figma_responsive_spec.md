# Earnalism Responsive Spec

## Viewports
- `1920 x 1080`
- `1536 x 864`
- `1440 x 900`
- `1366 x 768`
- `1280 x 800`
- `1024 x 768`
- `820 x 1180`
- `430 x 932`
- `390 x 844`

## Requirements
- No horizontal overflow.
- No clipped covers.
- CTA visible and keyboard accessible.
- Reader settings usable on mobile.
- Audio controls never appear for unapproved titles.
- Bengali prose keeps generous line-height.
- Library controls wrap without crowding.
