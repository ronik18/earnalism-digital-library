# Home Page 10/10 Spec

The Home page must establish Earnalism as a calm bilingual literary sanctuary.

Requirements:
- Compact calm header.
- Hybrid editorial hero.
- Dracula must not be the main brand hero.
- Three curated action cards: Bengali Classics, English Classics, Approved Audiobooks.
- Bengali classics are visible and premium.
- Approved audiobook CTAs render only from release evidence.
- Reader-only books feel complete, not incomplete.
- Smaller calm typography; no claustrophobic hero.
- Graphical cover tiles; no typographic-only public covers.
- No horizontal overflow.
- No unapproved audio CTA.
