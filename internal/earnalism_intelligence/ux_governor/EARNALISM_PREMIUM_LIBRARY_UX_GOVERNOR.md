# Earnalism Premium Library UX Governor

Earnalism UX must feel like a premium literary sanctuary for Bengali and English classics. It must be calm, beautiful, trustworthy, fast, accessible, conversion-ready, and release-gate truthful.

Every pixel must improve one of: trust, orientation, reading comfort, listening comfort, brand memory, conversion, accessibility, or performance.

Do not make Earnalism look like generic SaaS.
Do not make Dracula the entire brand.
Do not use huge claustrophobic typography.
Do not use typographic-only public covers.
Do not expose unapproved audiobooks.

## Phase Rule

Interactive UX work proceeds one phase at a time. HOME is approved for the current sprint. LIBRARY and later phases require a new owner approval record.
