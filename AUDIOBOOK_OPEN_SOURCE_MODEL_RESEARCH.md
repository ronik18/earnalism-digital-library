# Open-Source Bengali Audiobook Model Research

Scope: `INTERNAL_REVIEW_ONLY`.

This research matrix supports a local bake-off for regenerated Bengali narration of **Kshudhita Pashan / The Hungry Stones**. It does not approve public audio, does not enable a Listen Now CTA, does not upload media, and does not call paid/cloud APIs.

## Launch Truth

- Dracula is the only live approved core reading title.
- Dracula audio remains disabled.
- Kshudhita Pashan remains pipeline-only.
- All generated samples are internal review artifacts until rights, license, quality, and owner review pass.

## License Policy

No candidate is commercially approved by this PR alone. Commercial readiness requires:

- code license review,
- pretrained weights license review,
- dataset/dependency license review,
- consent review for any reference voice workflow,
- written owner approval before any local generation,
- separate public release approval after listening QA.

## Preferred Benchmark Order

1. Svara-TTS v1 for expressive Bengali Gothic style, if locally installable and license checks pass.
2. MahaTTS/MahaTTSv2 for commercial-readiness and naturalness comparison.
3. AI4Bharat Indic-TTS for pronunciation baseline.
4. F5-TTS only after license and consented reference voice governance.

## Explicit Non-Actions

- No model was installed.
- No model weights were downloaded.
- No audio was generated by default.
- No public URL was created.
- No cloud TTS/STT/LLM/audio API was called.
- No public audiobook or Listen Now CTA was enabled.
